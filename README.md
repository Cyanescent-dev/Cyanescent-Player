# Cyanescent Player


A Free Next Generation Audio Visualiser and Music Video Creator. Cyanescent Player is a native macOS music player and audio-reactive visualiser built for immersive generative live visuals, adaptive preset journeys and video creation. It also supports input from Pocket Scion, its it's special Bio-Feedback AV Synthesis Mode. 

## Cyanescent Player 0.9 Preview

The first public preview includes:

- 20 curated audio-reactive visual presets
- Adaptive and shuffle visual journeys
- Smooth preset transitions
- Preset playlists and management
- Custom preset importing
- Scion input and synthesis modes
- Music playback
- Video export up to 1080p
